# FBSD-Desktop

###  Про FBSD-Desktop

FBSD-Desktop - это bash скрипт для настроки графисеского окружения пользовател из голой FreeBSD! 


### Установка

```
% sudo pkg install bash git 
% git clone https://github.com/eleutherius/fbsd-desktop
% cd fbsd-desktop
% sudo bash fbsd-desktop.sh 
```
